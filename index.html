    Thoughts on Progressive Enhancemen

    aditional source

    Progressive enhancement. Is a philosophy that says that the content should be the highest priority. Even if it doesn't have all the bells and whistles, delivering the content is the most important thing. This is achieved by having your base content in your html and your style and javascript linked and in other pages. I presume this allows websites to run the html without any errors. I also allow the page to load faster, because the important content gets pulled up immediately. All of this will increase accessibility of your website. Meaning more people will be able to use it.
